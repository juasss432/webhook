How to Use

1. Create a Webhook

Go to the "Create" tab and give your webhook a name. You'll receive a unique URL.

2. Send Data to Your Webhook

Use any HTTP client to POST data to your webhook URL:

curl -X POST https://your-webhook-url \
-H "Content-Type: application/json" \
-d '{"message": "Hello from my app!"}'

3. JavaScript Example

fetch('https://your-webhook-url', {
method: 'POST',
headers: { 'Content-Type': 'application/json' },
body: JSON.stringify({ message: 'Test' })
});

4. Monitor Requests

Click "View Logs" in the "Manage" tab to see all incoming requests to your webhook in real-time.
